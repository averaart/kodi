# Kodi

Ruby wrapper for [Kodi JSON-RPC API](http://kodi.wiki/view/JSON-RPC_API/v6).


## Installation

Add this line to your application's Gemfile:

    gem 'kodi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kodi
